# Calculator-react

Калькулятор, созданный на React в учебных целях.
Базовый проект для отработки навыков работы с React-компонентами, хуками и логикой вычислений.

Функционал:
```
Предполагаемый функционал (исходя из типа проекта):
Базовые арифметические операции: сложение, вычитание, умножение, деление.
Интерфейс с кнопками и дисплеем.
Обработка пользовательского ввода.
```
Технологии
```
React — библиотека для построения интерфейсов.
Create React App — инструмент для быстрого старта проекта.
JavaScript (ES6+) — логика приложения.
CSS — стилизация компонентов.
```
